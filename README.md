# Mine Risk Map — Kharkiv Oblast

Інтерактивна карта для візуалізації сесій інформування населення про ризики вибухонебезпечних предметів та безпечнішу поведінку в Харківській області.

Проєкт поєднує:
- власну веб-форму для внесення нових сесій,
- Google Sheets як сховище даних,
- Google Apps Script як легкий API,
- GitHub Pages як хостинг,
- Leaflet як основу карти,
- heatmap і кластери для аналітичного відображення.

---

## 1. Призначення проєкту

Цей проєкт створений для того, щоб:

- збирати дані про проведені MRE-сесії в єдиному форматі,
- автоматично прив’язувати сесії до населених пунктів Харківської області,
- відображати точки проведення на карті,
- будувати heatmap за кількістю бенефіціарів,
- фільтрувати дані за періодом, інструктором, районом і громадою,
- експортувати поточний вигляд карти в PNG для звітів, презентацій і донорських матеріалів.

---
